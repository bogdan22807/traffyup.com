TRAFFYUP.com — прийом оплати через Monobank
============================================

⚠️ РОБОЧА КОПІЯ ПРОЄКТУ: усі зміни робити ТІЛЬКИ в цій папці.
   Шлях: /Users/alinaozhinskaya/Desktop/TRAFFYUP.com

Що в папці:
  • index.html      — головна (редирект на оплату)
  • pricing.html    — сторінка вибору послуги та кнопка «Оплатити»
  • server.js       — сервер (роздає сайт + створює інвойси Monobank)
  • .env            — ваш токен Monobank (вже прописаний)
  • api/            — логіка створення інвойса

Як приймати оплату:
  1. Двічі клікніть по файлу: ЗАПУСТИТЬ_СЕРВЕР.command
  2. Відкрийте в браузері: http://localhost:3000
  3. Виберіть платформу, послугу, кількість → натисніть «Оплатити через Monobank»
  4. Клієнта перекине на сторінку Monobank — після оплати гроші прийдуть на ваш рахунок ФОП

Якщо Node.js не встановлений — завантажте з https://nodejs.org (версія LTS).

---
Vercel: якщо сайт не відкривається
-----------------------------------
1. Vercel Dashboard → ваш проєкт → Settings → General:
   • Root Directory — порожньо або . (корінь репо)
   • Framework Preset — Other (не Next.js, не Create React App)
2. Deployments: останній деплой має бути Ready (зелена галочка). Якщо Failed — відкрийте лог і виправте помилку.
3. Відкривайте саме домен проєкту: https://ваш-проєкт.vercel.app (без /api та без шляху до папки).
4. Якщо відкривається порожня сторінка — перевірте консоль браузера (F12 → Console) на помилки.
