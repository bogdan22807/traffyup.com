# Мобильная версия TRAFFYUP

В этой папке лежит стили для телефонов: **mobile.css**.

Они уже подключены на всех страницах сайта (index, pricing, orders, affiliate) и работают автоматически на экранах до 768px.

## Что исправлено на телефонах

- Нет горизонтального скролла и наложений
- Шапка и меню не перекрывают друг друга, баланс не обрезается
- Кнопки и ссылки не меньше 44–48px по высоте (удобно нажимать)
- Блоки статистики и карточки идут в один столбец
- Модальное окно и формы на всю ширину экрана
- Учтены вырезы экрана (safe area)

## Как выложить в интернет

Залей папку **mobile** вместе с остальным сайтом. Путь к файлу должен быть: `твой-сайт.ru/mobile/mobile.css`. Тогда всё будет работать без изменений.
